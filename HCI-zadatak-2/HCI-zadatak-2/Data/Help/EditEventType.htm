Edit event type

This tab is used for editing the tag that is selected inside the Event types -> View tab.

Information that can be edited (note - all fields are required):

  • Name - enter a name for the event type.
  • Icon - select a picture to serve as the icon of the event.
  • Description - enter a description for the event type.

By left-clicking the Save changes button at the bottom of the edit form, the changes made to the event type will be saved.
