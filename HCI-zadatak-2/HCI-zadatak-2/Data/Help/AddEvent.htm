Add event

This tab is used for adding a new event.

Information that needs to be input (note - all fields are required):

  • Id - enter a character sequence that uniquely identifies the event.
  • Name - enter a name for the event.
  • Description - enter a description of the event.
  • Event type - choose an event type for the event.
  • Alcohol serving category - choose an alcohol serving category for the event.
  • Icon - choose an icon for the event.
  • Suitable for handicapped - choose whether the event is suitable for handicapped or not.
  • Smoking allowed - choose whether smoking is allowed for the event or not.
  • Outdoors - choose whether the event takes place outdoors or not.
  • Price category - choose a price category for the event.
  • Expected audience (numeric) - enter an expected audience for the event.
  • Date - choose a date for the event.
  • Tags - choose tags for the event.

By left-clicking the Create event button at the bottom of the add form, an event with the entered data will be created.
