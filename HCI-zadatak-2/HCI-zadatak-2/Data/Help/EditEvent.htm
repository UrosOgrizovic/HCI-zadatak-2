Edit event

This tab is used for editing the tag that is selected inside the Events -> View tab.

Information that may be edited (note - all fields are required):

  • Name - enter a name for the event.
  • Description - enter a description for the event.
  • Event type - choose a single event type for the event.
  • Alcohol serving category - choose a single alcohol serving category for the event.
  • Icon - choose a picture to serve as the icon of the event.
  • Suitable for handicapped - choose whether the event is suitabale for handicapped or not.
  • Smoking allowed - choose whether the event allows smoking or not.
  • Outdoors - choose whether the event is outdoors or not.
  • Price category - choose a price category for the event.
  • Expected audience - enter the expected audience for the event.
  • Date - choose when the event takes place.
  • Tags - add/remove tags to the event.

By left-clicking the Save changes button at the bottom of the edit form, the changes made to the event will be saved.
