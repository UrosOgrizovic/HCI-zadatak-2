Add event type

This tab is used for adding a new event type.

Information that needs to be input (note - all fields are required):

  • Id - enter an id for the event. This id is used to uniquely identify the event type.
  • Name - enter a name for the event type.
  • Icon - choose a picture to serve as the icon of the event type.
  • Description - enter a description for the event type.

By left-clicking the Create event type button at the bottom of the add form, an event type with the entered data will be created.
