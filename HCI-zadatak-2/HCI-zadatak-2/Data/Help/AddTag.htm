Add tag

This tab is used for adding a new tag.

Information that needs to be input:

  • Id - a character sequence that is unique for every tag
  • Color - choose a color for the tag that you want to create
  • Description - a textual description of the tag that you want to create

By left-clicking the Create tag button at the bottom of the add form, a tag with the entered data will be created

Please bear in mind that all fields are required.
