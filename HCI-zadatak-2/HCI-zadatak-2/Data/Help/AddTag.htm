Add tag

This tab is used for adding a new tag.

Information that needs to be input (note - all fields are required):

  • Id - enter a character sequence that uniquely idenitifes the tag.
  • Color - choose a color for the tag.
  • Description - enter a textual description of the tag.

By left-clicking the Create tag button at the bottom of the add form, a tag with the entered data will be created.
